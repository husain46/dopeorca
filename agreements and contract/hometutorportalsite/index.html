1
Scope of Work — What Will Be Built

DopeOrca will design, develop, and deploy a complete portal for bombayhometutors.com:

  • Homepage — Full landing page: hero, how it works, subjects, areas, footer
  • Find Tutors Page — Filterable tutor directory
  • Post Requirement Page — Student form with subject, class, area, budget, contact
  • Login & Register — Secure auth for students, tutors, and admin
  • Tutor Dashboard — Profile, browse masked requirements, Razorpay ₹500 unlock flow
  • Student Dashboard — Post and manage requirements
  • Razorpay Integration — Rs. 500 per contact unlock (UPI, cards, net banking)
  • Admin Panel — Approve tutors, manage users, view all payments
  • Database + Deployment — MongoDB Atlas + live on Vercel with domain
⚠️ Anything not listed above is not included. Extra features = separate quote.
